# Jeni Patel | Cloud & DevOps Architect Portfolio

This is the source code for my personal portfolio website, designed to showcase my experience and projects in the Cloud and DevOps space.

## 🚀 About the Project

Built with **React**, **Vite**, and **Vanilla CSS**, this portfolio highlights my journey as a DevOps professional, featuring interactive project cards, a professional timeline, and a deep dive into my technical skill set.

## 🏗️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
